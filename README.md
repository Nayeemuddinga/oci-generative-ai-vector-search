# OCI Generative AI – Vector Search Demo

This repository demonstrates how to build an AI-powered semantic search solution using
Oracle Cloud Infrastructure (OCI) Generative AI embeddings and Oracle Autonomous Database
vector capabilities.

## Architecture Overview

- OCI Generative AI – Embeddings
- Oracle Autonomous Database (ATP)
- Python
- Vector similarity search using cosine distance

## Use Case

Enable semantic search over enterprise documents where users can search by meaning,
not keywords.

Example:
"Find documents related to database security compliance"

## Components

1. Document ingestion
2. Embedding generation using OCI Generative AI
3. Vector storage in Autonomous Database
4. Semantic similarity search

## Prerequisites

- OCI account
- Autonomous Database (ATP)
- Python 3.9+
- OCI API Key configured
